# Current Important Decisions

## Accepted

- MIDI-first architecture preferred.

- Practice and performance are distinct modes.

- Performance should not feel like engineering.

- Open audio ecosystems are strategically important.

- Textual/configurable infrastructure is preferred.

- Carla is currently the leading host candidate.

- OSC control is strategically attractive.

- CI/testing should exist from early in the project.

## Rejected or deprioritised

- Heavy DAW-centric live workflows.

- MainStage/Logic as long-term programmable backend.

- Audio-first looping as primary architecture.

- Laptop-screen-centric performance aesthetics.
