# Music Programming Project Context

## Project Aim

Build a programmable music system for composition, practice, and live performance that preserves the immediacy and physicality of musical performance while enabling sophisticated computational control beneath the surface.

The system should feel like playing instruments, not operating software.

The project sits at the intersection of:

- live instrumental performance;

- programmable musical systems;

- symbolic music representation;

- reproducible engineering;

- configurable sound design;

- AI-assisted tooling;

- open audio infrastructure.
